# Installation
1. Clone the git repo
```sh
git clone https://github.com/CHARANCHARY/get-crypto-details.git
```
2. Install dependencies
```sh
npm install
```

3. Start the backend
```sh
cd backend
npm rum start
```
4. Start the frontend
```sh
cd frontend
npm rum start
```
